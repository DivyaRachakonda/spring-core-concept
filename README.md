# Spring Core Concepts

This repository contains simple Java projects demonstrating fundamental concepts of the Spring Framework.
These examples are created using Maven and focus on understanding how Spring manages objects and application flow.

---

## 📌 Projects Included

### 1. Annotations-Based Configuration

* Demonstrates usage of annotations like `@Component`, `@Autowired`
* Shows how Spring automatically manages beans

### 2. Metadata Configuration

* Explains how Spring reads configuration metadata
* Demonstrates use of configuration classes and bean definitions

### 3. Bean Lifecycle

* Shows initialization and destruction of beans
* Demonstrates lifecycle methods and annotations

---

## ▶️ How to Run

1. Import the project into Eclipse as a Maven project
2. Update Maven dependencies
3. Run the main class (`App.java`)
4. Observe the output in console
